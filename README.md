# amstrad-diagnostics
A suite of diagnostic tests for the Amstrad CPC, Amstrad Plus and Amstrad GX4000.

It performs several diagnostics:
* Lower 64KB RAM
* Upper RAM
* System ROMs
* Keyboard 
* Detect CRTC model
